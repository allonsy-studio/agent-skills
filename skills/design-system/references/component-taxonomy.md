# Component Taxonomy

A catalog of established UI component patterns organized by function. Use
this to identify what kind of component the user is describing, find the
canonical name for it, and understand its relationship to other patterns.

This taxonomy is informed by The Component Gallery (component.gallery) and
cross-referenced against major design systems (Material, Spectrum, Carbon,
Lightning, Primer, Polaris, GOV.UK, USWDS).

---

## Sections

1. [Actions & Inputs](taxonomy/01-actions-inputs.md) — Buttons, links, and other elements that trigger actions or navigate.
2. [Navigation & Wayfinding](taxonomy/02-navigation-wayfinding.md) — Nav bars, breadcrumbs, tabs, pagination, and other orientation patterns.
3. [Disclosure & Layering](taxonomy/03-disclosure-layering.md) — Accordions, modals, drawers, popovers, tooltips, and dropdown menus.
4. [Feedback & Status](taxonomy/04-feedback-status.md) — Alerts, toasts, banners, progress bars, spinners, skeletons, and badges.
5. [Layout & Containers](taxonomy/05-layout-containers.md) — Cards, headers, footers, dividers, and layout grids.
6. [Data Display](taxonomy/06-data-display.md) — Tables, lists, trees, tags/chips, charts, avatars, plus notes on icons and empty states.
7. [Form Controls](taxonomy/07-form-controls.md) — Text inputs, selects, comboboxes, checkboxes, radios, toggles, sliders, datepickers, file uploads, and steppers.
8. [Media & Imagery](taxonomy/08-media-imagery.md) — Carousels and image/figure patterns.
9. [Utilities](taxonomy/09-utilities.md) — Visually-hidden text, skip links, and ARIA live regions.
10. [Application Chrome & Layout](taxonomy/10-application-chrome-layout.md) — App shells, split panes, sticky headers, toolbars, status bars, and notification centers.
11. [Application Tools](taxonomy/11-application-tools.md) — Panels, color pickers, and complex dropdown menus for SPA tool UIs.
12. [Composite Patterns](taxonomy/12-composite-patterns.md) — Higher-level patterns assembled from multiple primitives (search, command palette, data table, etc.).
