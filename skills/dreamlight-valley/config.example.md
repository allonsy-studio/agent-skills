# Dreamlight Valley preferences

Drop a copy of this file at one of the paths listed in SKILL.md's
"Configuration" section. Everything is optional — delete the lines you don't
care about and the skill will use its built-in defaults for them.

## Floors

How many of each item to keep on hand before `/cook` will spend it, and the
target `/restock` gathers up to.

- Default floor: **50**
- Collectables floor (ores, dust, wood, materials): **99**

## Cooking

- Default objective: **profit** _(or `energy`)_
- Top recipes to show: **5**
- Show "highest single payout" callout: **yes**

## Recipe lookups

- Extra fields to show alongside the ingredient list: _(none)_
  _(any of: `star_rating`, `course`, `energy`, `sale_price` — comma-separated)_

## Restock checklist

- Include items not in the catalog: **yes**
- Show items already at floor: **no**
