# /recipe — Identify a dish

Enter the `dreamlight-valley` skill in **Identify mode**. Read
`references/identify.md` for the full procedure, then apply it to:

- `$ARGUMENTS` (typed dish name, optional)
- any attached recipe card, cookbook page, or in-game menu screenshot

Also load the user's config per SKILL.md's "Configuration" section before
answering — it may set `recipe.extra_fields` to include star rating, energy,
or sell price by default.
